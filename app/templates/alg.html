How to use the LWE bit encryption protocol module
  • Select the matrix dimension, number of rows will be chosen by a recommended rule.
  • Chose a proper prime modulus, recommendation is between n^2 and 2n^2, or just experiment.
  • Then click on Generate parameters to obtain the key pair used in the encryption.
  • Now you can select which bit you want to encrypt, and then click on Encrypt the message.
  • You can see how the encryption is being calculated on the left of the controls.
  • After you have encrypted the message, you can try to decrypt it using the Decrypt the message button.
  • You can see under what condition is the message decrypted.
  • You can also observe how close the message is to the decryption point, which is "0" in the uppermost point and "1" in the lowermost point on the unit circle.
If you want to see the whole matrices, click on download to obtain the parameters in JSON format!
When you decrypt a wrong bit, try selecting larger dimension and/or larger prime.
